لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە یەک دێڕ لە فایلەکەوە دەگەڕێنێتەوە، بنووسە: read line.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە لیستی دێڕەکان لە فایلەکەوە دەگەڕێنێتەوە، بنووسە: دێڕەکان بخوێنەرەوە.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە شوێنی پەڕگەکە دەگۆڕێت، بنووسە: seek.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە دەگەڕێنێتەوە کە ئایا فایلەکە ڕێگەمان پێدەدات شوێنی پەڕگەکە بگۆڕین، بنووسە: seeable.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە شوێنی ئێستای فایلەکە دەگەڕێنێتەوە، بنووسە: tell.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە قەبارەی پەڕگەکە دەگۆڕێت بۆ قەبارەیەکی دیاریکراو، بنووسە: truncate.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە دەگەڕێنێتەوە کە ئایا دەتوانرێت فایلەکە بۆی بنووسرێت یان نا، بنووسە: writable.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە ڕیزەکەی دیاریکراو دەنووسێت بۆ فایلەکە، بنووسە: بنووسە.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە لیستی ڕیزەکان بۆ فایلەکە دەنووسێت، بنووسە: دێڕەکان بنووسە.
لە پایتۆن بۆ بەکارهێنانی دەربڕینەکە بۆ ئۆپەراتۆرێکی لۆژیکی، بنووسە: و.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە ناوی نادیار دروست دەکات، بنووسە: as.
لە پایتۆن بۆ بەکارهێنانی دەربڕینەکە بۆ چاککردنەوە، بنووسە: assert.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە لە لوپێکەوە دەشکێت، بنووسە: break.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە پۆلێک پێناسە دەکات، بنووسە: class.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە بەردەوام دەبێت بۆ دووبارەکردنەوەی داهاتووی لوپێک، بنووسە: بەردەوام بە.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە فەنکشنێک پێناسە دەکات، بنووسە: define.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە ئۆبجێکتی دەسڕێتەوە، بنووسە: delete.
لە پایتۆندا، بۆ بەکارهێنانی دەربڕینەکە بۆ ڕستەی مەرجدار، هەمان شت وەک else if، بنووسە: else if.
لە پایتۆن بۆ بەکارهێنانی دەربڕینەکە بۆ ڕستەی مەرجدار، بنووسە: else.
لە پایتۆن بۆ بەکارهێنانی دەربڕینەکە بۆ ئیستسناءەکان، چی بکەین کاتێک ئیستسنای ڕوودەدات، بنووسە: except.
لە پایتۆندا، بۆ بەکارهێنانی دەربڕینەکە بۆ بەهایەکی بولی، کە ئەنجامی کارەکانی بەراوردکردنە، بنووسە: هەڵە.
لە پایتۆندا بۆ بەکارهێنانی ئەو دەربڕینەی کە بەکاردێت لەگەڵ ئیستسناء، بلۆکێکی کۆد کە جێبەجێ دەکرێت گرنگ نییە ئیستسناء هەبێت یان نا، بنووسە: لە کۆتاییدا.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە for loop دروست دەکات، بنووسە: for.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە بەشە تایبەتەکانی مۆدیولێک هاوردە دەکات، بنووسە: from.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە گۆڕاوێکی جیهانی ڕادەگەیەنێت، بنووسە: جیهانی.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە ڕستەیەکی مەرجدار دروست دەکات، بنووسە: if.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە مۆدیولێک هاوردە دەکات، بنووسە: import.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە پشکنین دەکات کە ئایا بەهایەک لە لیست و tuple و هتددا هەیە یان نا، بنووسە: in.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە تاقی دەکاتەوە ئەگەر دوو گۆڕاو یەکسان بن، بنووسە: is.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە فەنکشنێکی بێناو دروست دەکات، بنووسە: lambda.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە نوێنەرایەتی بەهایەکی پووچ دەکات، بنووسە: هیچ.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە گۆڕاوێکی نالۆکاڵی ڕادەگەیەنێت، بنووسە: non-local.
لە پایتۆن بۆ بەکارهێنانی دەربڕینەکە بۆ ئۆپەراتۆرێکی لۆژیکی، بنووسە: نەخێر.
لە پایتۆن بۆ بەکارهێنانی دەربڕینەکە بۆ ئۆپەراتۆرێکی لۆژیکی، بنووسە: یان.
لە پایتۆن بۆ بەکارهێنانی دەربڕینەکە بۆ ڕستەیەکی پووچ، ڕستەیەک کە هیچ ناکات، بنووسە: pass.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە ئیستسنای بەرز دەکاتەوە، بنووسە: raise exception.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە لە فەنکشنێک دەردەچێت و بەهایەک دەگەڕێنێتەوە، بنووسە: return.
لە پایتۆندا، بۆ بەکارهێنانی دەربڕینەکە بۆ بەهایەکی بولی، ئەنجامی کارەکانی بەراوردکردن، بنووسە: ڕاستە.
لە پایتۆندا، بۆ بەکارهێنانی ئەو دەربڕینەی کە try...except statement دروست دەکات، بنووسە: try.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە while loop دروست دەکات، بنووسە: while.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە مامەڵەکردن لەگەڵ ئیستسناء ئاسان دەکات، بنووسە: لەگەڵ.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە کۆتایی بە فەنکشنێک دەهێنێت، مۆلیدێک دەگەڕێنێتەوە، بنووسە: yield.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک لە حیساباتی ژمارەییدا ڕوودەدات، بنووسە: هەڵەی ژمێریاری.
لە پایتۆندا، بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک ڕستەیەکی assert شکست دەهێنێت، بنووسە: Assertion Error.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک ئاماژە یان تەرخانکردنی تایبەتمەندییەک شکستی دەهێنێت، بنووسە: هەڵەی تایبەتمەندی.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە پۆلێکی بنەڕەتییە بۆ هەموو ئیستسناءەکان، بنووسە: ئیستسناء.
لە پایتۆندا، بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک شێوازی input() بەر مەرجی "کۆتایی پەڕگە" (EOF) دەدات، بنووسە: End Of File Error.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک حیسابکردنی خاڵی هەڵدەفڕێت شکست دەهێنێت، بنووسە: هەڵەی خاڵی هەڵدەفڕێت.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک مۆلیدێک داخراوە (بە شێوازی close())، بنووسە: Generator Exit.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک مۆدیولێکی هاوردەکراو بوونی نییە، بنووسە: هەڵەی هاوردەکردن.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک ئیندێنتکردن ڕاست نییە، بنووسە: Indentation Error.
لە پایتۆندا بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک ئیندێکسێکی زنجیرەیەک بوونی نییە، بنووسە: Index Error.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک کلیلێک لە فەرهەنگێکدا بوونی نییە، بنووسە: Key Error.
لە پایتۆندا، بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک دروست دەکات کاتێک بەکارهێنەر Ctrl+c، Ctrl+z یان Delete دەکات، بنووسە: Keyboard Interrupt.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک هەڵەکانی وروژێنراو نادۆزرێتەوە، بنووسە: هەڵەی گەڕان.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک دروست دەکات کاتێک بەرنامەیەک بیرگەی تەواو دەبێت، بنووسە: هەڵەی بیرگە.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک گۆڕاوێک بوونی نییە، بنووسە: Name Error.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک میتۆدێکی ئەبستراکت پێویستی بە پۆلێکی بۆماوەیی هەیە بۆ ئەوەی میتۆدەکە جێبەجێ بکات، بنووسە: هەڵەی جێبەجێ نەکراوە.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک کارێکی پەیوەندیدار بە سیستەم دەبێتە هۆی هەڵە، بنووسە: هەڵەی OS.
لە پایتۆندا بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک ئەنجامی حیسابکردنی ژمارەیی زۆر گەورەیە، بنووسە: Overflow Error.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک ئۆبجێکتی ئاماژەی لاواز بوونی نییە، بنووسە: هەڵەی ئاماژە.
لە پایتۆندا بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک هەڵەیەک ڕوودەدات کە سەر بە هیچ ئیستسنایەکی تایبەت نییە، بنووسە: هەڵەی کاتی جێبەجێکردن.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک شێوازی next() ی دووبارەکەرەوەیەک بەهای زیاتری نییە، بنووسە: Stop Iteration.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک هەڵەی ڕستەسازی ڕوودەدات، هەڵەی ڕستەسازی بنووسە.
لە پایتۆندا بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک ئیندێنتاسیۆن لە تاب یان بۆشایی پێکهاتووە، بنووسە: هەڵەی تاب.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک هەڵەی سیستەم ڕوودەدات، بنووسە: System Error.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک فەنکشنی sys.exit() بانگ دەکرێت، بنووسە: System Exit.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک دوو جۆری جیاواز یەکدەگرن، بنووسە: Type Error.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک گۆڕاوێکی ناوخۆیی پێش تەرخانکردن ئاماژەی پێدەکرێت، بنووسە: هەڵەی ناوخۆیی نەبەستراو.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک دروست دەکات کاتێک کێشەی یونیکۆد ڕوودەدات، بنووسە: Unicode Error.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک کێشەی کۆدکردنی یونیکۆد ڕوودەدات، بنووسە: هەڵەی کۆدکردنی یونیکۆد.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک دروست دەکات کاتێک کێشەی کۆدکردنەوەی یونیکۆد ڕوودەدات، بنووسە: هەڵەی دیکۆدی یونیکۆد.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک دروست دەکات کاتێک کێشەی وەرگێڕانی یونیکۆد ڕوودەدات، بنووسە: هەڵەی وەرگێڕانی یونیکۆد.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک بەهایەکی هەڵە لە جۆرێکی داتا دیاریکراودا هەیە، بنووسە: هەڵەی بەها.
لە پایتۆن بۆ بەکارهێنانی ئەو دەربڕینەی کە هەڵەیەک بەرز دەکاتەوە کاتێک ئۆپەراتۆری دووەم لە دابەشکردنێکدا سفرە، بنووسە: هەڵەی دابەشکردنی سفر.
